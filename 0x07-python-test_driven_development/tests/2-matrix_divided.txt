==============================
How to use 2-matrix_divided.py
==============================

this module define a function that divides all elements of a matrix.

Usage
=====

``matrix_divided(matrix, div)``` return a new matrix with the divided value.

    case 1: dividing list by int number.
    ::

        >>> matrix_divided = __import__('2-matrix_divided').matrix_divided
        >>> matrix = [[1, 2, 3],[4, 5, 6]]
        >>> print(matrix_divided(matrix, 3))
        [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
        >>> print(matrix)
        [[1, 2, 3], [4, 5, 6]]

    ::

    case 2: dividing list by float number.
    ::

        >>> print(matrix_divided(matrix, 3.3))
        [[0.3, 0.61, 0.91], [1.21, 1.52, 1.82]]

    ::

    case 3: dividing number by 0.
    ::

        >>> print(matrix_divided(matrix, 0))
        Traceback (most recent call last):
        ZeroDivisionError: division by zero

    ::

    case 4: the list contain a string element.
    ::

        >>> matrix = [[1, '2', 3],[4, 5, 6]]
        >>> print(matrix_divided(matrix, 3.3))
        Traceback (most recent call last):
        TypeError: matrix must be a matrix (list of lists) of integers/floats

    ::

    case 5: the rows aren't the same type.
    ::

        >>> matrix = [[1, 2],[4, 5, 6]]
        >>> print(matrix_divided(matrix, 3.3))
        Traceback (most recent call last):
        TypeError: Each row of the matrix must have the same size
